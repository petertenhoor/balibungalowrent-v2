Instantly subscribe guests to Mailchimp lists to send stylish emails with significant booking data.

== Description ==

Instantly connect Mailchimp with your hotel website to send stylish emails with significant booking data, helpful automated messages and targeted marketing campaigns.

== Installation ==

1. Upload the MotoPress plugin to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
